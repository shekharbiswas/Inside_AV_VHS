# Autonomous Vehicles with Python 

## Course Modules

### Module 1: Python Basics & Data for AVs

Learn Python step by step with variables, loops, and functions, then move into **pandas** and **matplotlib** to analyze driving datasets. By the end, you'll be able to load real data such as speed and GPS logs, visualize it, and understand why Python is the foundation of modern autonomous vehicle systems.



### Module 2: How Self-Driving Cars See the World

Discover how autonomous vehicles use sensors such as cameras, LiDAR, and GPS to interpret their surroundings.

Using **OpenCV**, you'll perform simple computer vision tasks like lane and vehicle detection, gaining a beginner-friendly understanding of how cars "see" the world around them.



### Module 3: Decision Making & Navigation Basics

Explore how autonomous vehicles make driving decisions by combining information from multiple sensors.

You'll implement simple driving rules, simulate obstacle detection, and create logic for scenarios such as stopping at traffic lights and turning at intersections. This module demonstrates how autonomous vehicles transform perception into action.



### Module 4: Simulation, Testing & Capstone Project

Learn why simulation is essential for autonomous vehicle development using platforms like **CARLA**.

Work with real driving datasets to visualize routes, identify detected objects, and complete a hands-on capstone project that combines data input, perception, and decision-making into a complete mini autonomous driving pipeline.



## By the End of This Course, You Will Be Able To

- Write Python programs using libraries such as **pandas**, **matplotlib**, and **OpenCV**.
- Understand how autonomous vehicles use sensors to perceive and interpret their environment.
- Build simple perception and decision-making logic for autonomous driving scenarios.
- Work with real-world datasets and basic simulations to create a mini autonomous vehicle pipeline.



## Requirements

- Basic computer knowledge.
- Familiarity with Python is helpful but not mandatory.
- Python **3.x** installed on your computer.
